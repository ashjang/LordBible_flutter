# LordBible_Flutter

## 프로젝트 개요

- KJV, NIV, 개역개정, KJV흠정역 제공하는 성경 어플리케이션

### 1. 주요 기능
- 4가지 버전의 성경말씀 제공 및 검색

#### 페이지 구조
- 홈(home)
    - 오늘의 말씀(랜덤)
    - 즐겨찾는 말씀
- 성경(bible)
    - 성경 말씀 버전
    - 말씀 즐겨찾기
    - 읽기 완료
    - 위젯 추가
- 검색(search)
    - 4가지 버전에서 특정 단어 검색 
- 설정(setting)
    - 읽은 말씀 목록
    - 성경 버전 선택
    - 글자 크기 조정
    - 다크 모드
- 위젯(widget)
    - 암송하고 싶은 성경 구절
